# Advanced AI Interview Copilot

A production-style AI interview preparation platform.

## Features
- AI-powered interview evaluation
- Whisper speech-to-text
- RAG architecture starter
- Streamlit analytics dashboard
- FastAPI backend
- Docker support
- CI/CD workflow

## Run Project

### Install
```bash
pip install -r requirements.txt
```

### Run Streamlit
```bash
cd streamlit_app
streamlit run app.py
```

### Run Backend
```bash
cd backend/api
uvicorn main:app --reload
```
